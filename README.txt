PROGAMMA un software per l'analisi dei dati provenienti dal software MAESTRO(c) (aquisizione di dati di rivelatori gamma, alfa e beta).


INSTALLAZIONE:
 Per compilare ed installare "proGamma" è sufficiente lanciare da terminale "make".
 
 Requisiti di sistema: 
	È necessario avere installato il framework ROOT CERN (https://root.cern.ch/).
	È necessario avere un compilatore che supporti il C++11 (gcc 4.8.1 o superiori).

UTILIZZO:
 Per informazioni su come usare "proGamma" lanciare "./proGamma --help"
